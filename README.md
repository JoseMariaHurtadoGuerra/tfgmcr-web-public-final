\# TFG MCR Web Public



Repositorio público asociado a la página web interactiva del Trabajo Fin de Grado:



\*\*Estudio de la estructura axial del nucleón en interacciones con neutrinos y antineutrinos\*\*



Autor: José María Hurtado Guerra  

Universidad de Sevilla  

Departamento de Física Atómica, Molecular y Nuclear



Este repositorio contiene únicamente los archivos necesarios para desplegar las aplicaciones públicas de Streamlit:



\- Formalismo teórico de la interacción débil cargada.

\- Factores de forma electromagnéticos e isovectores.

\- Factores de forma axial y pseudoescalar.

\- Secciones eficaces y comparación con datos de MINERvA en hidrógeno.
